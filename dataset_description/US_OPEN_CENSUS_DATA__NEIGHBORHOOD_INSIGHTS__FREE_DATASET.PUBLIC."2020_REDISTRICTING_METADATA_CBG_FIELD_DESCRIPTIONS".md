Table: US_OPEN_CENSUS_DATA__NEIGHBORHOOD_INSIGHTS__FREE_DATASET.PUBLIC."2020_REDISTRICTING_METADATA_CBG_FIELD_DESCRIPTIONS"

Description: Metadata mapping table for 2020 redistricting Census Block Group (CBG) datasets. Use this table when you need to interpret coded column IDs into human-readable field names, topics, and universes for redistricting-era CBG data.
---
Columns:
FIELD_NAME: Human-readable field name/label for the metric.
COLUMN_ID: Encoded column identifier used in CBG data tables.
COLUMN_TOPIC: Topical category of the metric (e.g., population, age, race).
COLUMN_UNIVERSE: Population universe the metric applies to.
