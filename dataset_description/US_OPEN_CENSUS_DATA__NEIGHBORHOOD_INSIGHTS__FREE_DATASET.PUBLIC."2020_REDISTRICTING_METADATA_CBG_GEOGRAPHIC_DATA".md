---
Table: US_OPEN_CENSUS_DATA__NEIGHBORHOOD_INSIGHTS__FREE_DATASET.PUBLIC."2020_REDISTRICTING_METADATA_CBG_GEOGRAPHIC_DATA"

Description: Geographic reference data for 2020 redistricting Census Block Groups (CBGs). Use this table when you need land/water area or centroid coordinates for a CBG, or when mapping/joining CBGs to geographic context.
---
Columns:
CENSUS_BLOCK_GROUP: Unique identifier for the census block group (primary key).
AMOUNT_LAND: Land area for the CBG.
AMOUNT_WATER: Water area for the CBG.
LATITUDE: Centroid latitude of the CBG.
LONGITUDE: Centroid longitude of the CBG.
