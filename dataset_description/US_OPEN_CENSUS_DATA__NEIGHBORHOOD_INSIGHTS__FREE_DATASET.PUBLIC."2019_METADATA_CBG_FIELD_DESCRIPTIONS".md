Table: US_OPEN_CENSUS_DATA__NEIGHBORHOOD_INSIGHTS__FREE_DATASET.PUBLIC."2019_METADATA_CBG_FIELD_DESCRIPTIONS"

Description: Metadata mapping table that explains the meaning of column codes in 2019 CBG (Census Block Group) datasets. Use this table to translate a coded field (e.g., B01001e1) into its human-readable definition, or to identify which topic/universe a column belongs to.
---
Columns:
TABLE_ID: Unique field code for a statistic in CBG tables (e.g., B01001e1).
TABLE_NUMBER: Census table identifier grouping related fields (e.g., B01001).
TABLE_TITLE: Human-readable title of the table containing the field.
TABLE_TOPICS: Broad topic category for the table (e.g., Age and Sex).
TABLE_UNIVERSE: Population universe the table covers (e.g., Total population).
FIELD_LEVEL_1: Top-level field descriptor (e.g., Estimate).
FIELD_LEVEL_2: Second-level descriptor within the table hierarchy.
FIELD_LEVEL_3: Third-level descriptor within the table hierarchy.
FIELD_LEVEL_4: Fourth-level descriptor within the table hierarchy.
FIELD_LEVEL_5: Fifth-level descriptor within the table hierarchy.
FIELD_LEVEL_6: Sixth-level descriptor within the table hierarchy.
FIELD_LEVEL_7: Seventh-level descriptor within the table hierarchy.
FIELD_LEVEL_8: Eighth-level descriptor within the table hierarchy.
FIELD_LEVELl_9: Ninth-level descriptor within the table hierarchy (note: column name includes a lowercase “l” in LEVELl).
FIELD_LEVEL_10: Tenth-level descriptor within the table hierarchy.
