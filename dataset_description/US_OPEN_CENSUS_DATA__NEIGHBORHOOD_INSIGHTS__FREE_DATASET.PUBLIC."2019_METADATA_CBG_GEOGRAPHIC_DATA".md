---
Table: US_OPEN_CENSUS_DATA__NEIGHBORHOOD_INSIGHTS__FREE_DATASET.PUBLIC."2019_METADATA_CBG_GEOGRAPHIC_DATA"

Description: Geographic reference data for 2019 census block groups. Use this table to map a CENSUS_BLOCK_GROUP to its land/water area and centroid latitude/longitude, or when you need geographic context for a block group.
---
Columns:
CENSUS_BLOCK_GROUP: Unique identifier for the census block group (primary key).
AMOUNT_LAND: Land area for the block group.
AMOUNT_WATER: Water area for the block group.
LATITUDE: Centroid latitude of the block group.
LONGITUDE: Centroid longitude of the block group.
